# Інструкція для розміщення готового файлу документації

## Крок 1: Згенеруйте шаблон документа

1. Тимчасово поверніть кнопку на генерацію шаблону (виклик `generateUserManual()`)
2. Завантажте згенерований DOCX файл
3. Відкрийте його в Microsoft Word

## Крок 2: Вставте зображення

У документі ви побачите червоні позначки з текстом:
```
>>> ТУТ ВСТАВИТИ РИСУНОК 1 <<<
>>> ТУТ ВСТАВИТИ РИСУНОК 2 <<<
...
>>> ТУТ ВСТАВИТИ РИСУНОК 15 <<<
```

Замініть кожну позначку відповідним зображенням:

- **Рисунок 2**: `src/image/image2.png` - Інтерфейс заповнення даних для місцевих доріг
- **Рисунок 3**: `src/image/image3.png` - Розрахунок нормативів для державних доріг
- **Рисунок 4**: `src/image/image4.png` - Розрахунок нормативів для місцевих доріг

## Крок 3: Збережіть готовий файл

1. Після вставки всіх зображень збережіть документ
2. Назвіть файл: **`Інструкція_користувача.docx`**
3. Покладіть цей файл у папку `public/docs/`

## Крок 4: Готово!

Тепер при натисканні кнопки "Завантажити інструкцію (DOCX)" користувачі будуть отримувати готовий файл з усіма зображеннями!

---

**Примітка**: Переконайтеся, що назва файлу точно `Інструкція_користувача.docx`, інакше система не зможе його знайти.
